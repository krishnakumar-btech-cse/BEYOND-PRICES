# 🌍 Overview

**HarvestPlanner** is an agricultural decision-support platform designed to help farmers choose **where and when to sell their crops**, while enabling consumers to identify **fresh and fairly priced produce**.

Agricultural markets often suffer from:

* price volatility
* poor logistics planning
* lack of market transparency
* post-harvest losses

HarvestPlanner addresses these challenges by combining **market data, logistics awareness, and AI-assisted reasoning** to guide better decisions across the agricultural supply chain.

The system supports **both farmers (producers)** and **consumers (buyers)** through a unified intelligence platform.

# 🧠 Platform Intelligence Experiences (15+)

HarvestPlanner delivers a suite of decision-support features across three primary domains.

---

# 🌾 Farmer Intelligence

### 1️⃣ Market Price Intelligence

Analyzes regional market pricing trends to identify favorable selling opportunities.

### 2️⃣ Best Market Decision Engine

Recommends the most profitable market by considering price trends, distance, and demand conditions.

### 3️⃣ Logistics Resilience Hub

Evaluates travel time, route conditions, and delays to ensure produce arrives before quality degradation.

### 4️⃣ Storage Advisory System

Provides recommendations on whether crops should be stored temporarily or sold immediately.

### 5️⃣ Market Stall Reservation Portal

Allows farmers to reserve mandi stalls digitally, ensuring guaranteed selling space before arrival.

### 6️⃣ Crop Status Monitoring

Tracks harvest windows and alerts farmers about potential quality downgrade risks.

### 7️⃣ Collective Transport Coordination

Enables nearby farmers selling similar crops to coordinate transport and reduce logistics costs.

### 8️⃣ AI Strategy Explanation

AI-generated reasoning explains why a particular market or time window is recommended.

---

# 🛒 Consumer Intelligence

### 9️⃣ Market Intelligence Feed

Shows live correlations between regional mandi supply strength and consumer market prices.

### 🔟 Smart Basket Optimization

Suggests optimal purchasing strategies across multiple markets to balance cost and freshness.

### 1️⃣1️⃣ Freshness Index

Scores produce based on harvest recency and estimated transport impact.

### 1️⃣2️⃣ Rational Purchase Insights

Visualizes supply density across regions to identify strong procurement zones.

# 🚀 Getting Started

## Prerequisites

* Node.js (v18 or later)
* Google Gemini API Key

---

## Installation

Clone the repository

```bash
git clone https://github.com/your-repo/harvest-planner.git
cd harvest-planner
```

Install dependencies

```bash
npm install
```

Create environment configuration

```env
VITE_GEMINI_API_KEY=your_api_key_here
```

Start the development server

```bash
npm run dev
```

The application will be available at:

```
http://localhost:5173
```
