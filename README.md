## 🚀 The Vision
**HarvestPlanner** is a next-generation "National Intelligence v2.0" platform designed to solve the structural inefficiencies in the agricultural supply chain. By leveraging Google Gemini AI and real-time market data, we empower farmers to maximize profits and consumers to access the freshest produce at the lowest prices.

## 🌟 15+ Core Intelligence Experiences

### 🌾 For the Farmer (Producer)
1.  **AI Price Forecasting**: Strategic analysis of crop pricing trends using LLM-driven intelligence.
2.  **Logistics Resilience Hub**: Real-time tracking of travel range, traffic delays, and transit quality loss.
3.  **Storage Advisory Engine**: Personalized recommendations for room vs. cold storage based on crop type.
4.  **Market Ethics Scoring**: Transparent "Trust Scores" for regional procurement nodes based on historical consistency.
5.  **Stall Reservation Portal**: Digital booking system for mandi stalls to ensure guaranteed procurement space.
6.  **Crop Status Monitoring**: Sentinel tracking of harvest windows and grade downgrade alerts.
7.  **Collective Intelligence**: Coordination tools for nearby farmers to optimize shared transport costs.
8.  **Automated Strategy Breakdown**: AI-generated justifications for "Best Market" vs "Best Time" decisions.

### 🛒 For the Consumer (Procurement)
9.  **Market Intelligence Feed**: Live correlation of regional mandi indices vs. retail price deltas.
10. **Smart Basket Optimization**: Algorithmic "Dual Market Splitting" to maximize savings while maintaining freshness.
11. **Freshness Indexing**: Data-backed scoring of produce based on harvest recency and transport impact.
12. **Rational Insights**: Brief, logical verdicts on whether to "Buy Now" or "Wait" for price stabilization.
13. **Regional Supply Heatmaps**: Visualizing the strength of regional supply nodes to identify pricing opportunities.

### 🌎 Systemic & Global Impact
14. **SDG Impact Tracking**: Real-time measurement of waste reduction (in KG) and alignment with UN Sustainability Goals.
15. **Low-literacy Design**: "Simple View" overrides providing ultra-concise, actionable advice for frontline users.
16. **Hybrid Architecture**: High-speed local mock data fallback for zero-latency demos in low-connectivity areas.
17. **Precision Navigation**: A premium, "National Architecture" UI designed for high-stakes decision making.

## 🚦 Getting Started

### Prerequisites
- Node.js (v18+)
- A Google Gemini API Key

### Installation

1. **Clone the Intelligence Engine:**
   ```bash
   git clone https://github.com/your-repo/harvest-planner.git
   cd harvest-planner
   ```

2. **Synchronize Dependencies:**
   ```bash
   npm install
   ```

3. **Configure Environment:**
   Create a `.env` file in the root directory:
   ```env
   VITE_GEMINI_API_KEY=your_actual_key_here
   ```

4. **Initialize Dev Server:**
   ```bash
   npm run dev
   ```
